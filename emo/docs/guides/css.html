CSS Styling

emo uses standard CSS syntax with dp/sp units.

Example

.container {
  background: #FFFFFF;
  padding: 24dp;
  spacing: 16dp;
}
.title {
  font-size: 28sp;
  font-weight: bold;
  color: #333333;
}

Using CSS in .em

<Column className="container">
  <Text className="title">Hello</Text>
</Column>
