Installation

emo runs on Linux, macOS, and Windows (amd64/arm64). Install it with a single command.

One-line install (recommended)

$ curl -fsSL https://raw.githubusercontent.com/crossberry-in/emo/main/install.sh | bash

This downloads a pre-built binary for your platform and installs it to ~/.local/bin/emo.

Install a specific version

$ curl -fsSL https://raw.githubusercontent.com/crossberry-in/emo/main/install.sh | bash -s -- v0.2.2

Install to a custom directory

EMO_INSTALL_DIR=/usr/local/bin curl -fsSL https://raw.githubusercontent.com/crossberry-in/emo/main/install.sh | bash

Authenticated install (avoids rate limits)

GITHUB_TOKEN=ghp_xxx curl -fsSL https://raw.githubusercontent.com/crossberry-in/emo/main/install.sh | bash

Build from source

git clone https://github.com/crossberry-in/emo
cd emo
go build -o /usr/local/bin/emo ./cmd/emo

Requires Go 1.22+.

Verify installation

emo --help
emo templates
emo components

Install the emo Go preview app

Download the APK from the emo-go releases page and install on your Android device:

adb install -r emo-go-v0.2.0.apk

Requirements

  • Go 1.22+ — for building from source
  • Android SDK + Platform Tools (adb) — for emo go
  • Android Studio or Gradle — for building the emo Go preview APK

💡 Tip: If emo is not found after install, add ~/.local/bin to your PATH:

echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.bashrc
source ~/.bashrc
