State & Hooks

emo uses React-like hooks for state management.

state declaration (in .em)

component App {
  state count = 0
  state name = "World"
  state enabled: bool = true

  render {
    <Text>{count}</Text>
    <Text>{name}</Text>
  }
}

State mutation

// This:
<Button onClick={() => count = count + 1}>+</Button>

// Becomes:
dsl.OnClick(func() { setCount(count + 1) })
